# DevWatchMan

## Project Overview
DevWatchMan is a real-time monitoring system designed to provide insights into your system's resources. Equipped with a FastAPI backend and an interactive web dashboard, DevWatchMan allows users to track CPU usage, memory consumption, disk activity, and more, all in a user-friendly interface.

## Features
- **Real-time Monitoring**: Get instantaneous updates on your system's performance.
- **FastAPI Backend**: Enjoy a lightweight, high-performance backend for your monitoring needs.
- **Interactive Dashboard**: Visualize data through a beautifully designed web interface.
- **Cross-Platform Compatibility**: Compatible with various operating systems including Windows, macOS, and Linux.

## Installation
To install and run DevWatchMan, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/Sakilalakmal/DevWatchMan.git
   ```
2. Navigate into the project directory:
   ```bash
   cd DevWatchMan
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Start the FastAPI server:
   ```bash
   uvicorn main:app --reload
   ```
5. Open your web browser and go to `http://127.0.0.1:8000` to access the dashboard.

## Usage
Once the server is running, you can monitor your system resources in real-time from the dashboard. The dashboard will automatically refresh to display the latest metrics.
